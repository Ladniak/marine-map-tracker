# Marine Map Tracker

## Опис програми

Marine Map Tracker — це вебзастосунок для візуалізації системи відстеження рухомих об’єктів у реальному часі.

Система отримує дані від backend API та відображає:

- станції спостереження (сенсори);
- азимутальні промені сигналів;
- об’єкти, визначені методом триангуляції;
- прогнозований напрямок руху об’єктів;
- параметри сигналів та сенсорів;
- інтерактивну карту з можливістю взаємодії.

Інтерфейс реалізований у вигляді адаптивної SPA (Single Page Application) з підтримкою мобільних пристроїв.

---

# Основні можливості

- Відображення станцій на інтерактивній карті;
- Відображення азимутальних напрямків сигналів;
- Відображення рухомих об’єктів;
- Вибір об’єкта або станції;
- Детальна інформація про об’єкти та сенсори;
- Інтерактивні popup-вікна;
- Автоматичне оновлення даних із backend API;
- Responsive UI для desktop та mobile;
- Інтеграція з REST API backend.
